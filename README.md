# 比赛获奖排名工具

一款轻量级、免注册、开箱即用的赛事获奖排名工具。面向知识竞赛、文体比赛、内部评比等短时评比场景，支持自定义奖项规则、快速录入选手成绩、自动完成排名计算与奖项匹配。全量数据本地存储，无服务端上传，移动端优先设计，用完即走。

## 特性

- 免注册开箱即用，无账号体系，不采集任何用户身份信息
- 移动端优先设计，兼容 PC / 平板，触摸交互友好
- 拖拽式奖项级别排序（PC 鼠标 + 移动端触屏长按）
- 同分并列名次顺延，奖项名额按级别从高到低自动匹配
- 同分跨奖项边界时整组顺延，公平分配
- 全量数据存储于浏览器 localStorage，刷新不丢失
- 极简现代 UI，原生 HTML/CSS/JS 实现，无第三方依赖

## 技术栈

- 后端：Flask（Python），仅提供静态页面路由与资源托管，无业务逻辑、无数据库
- 前端：原生 HTML5 + CSS3 + JavaScript（ES6+），无任何第三方框架
- 存储：浏览器 localStorage

## 环境依赖

- Python 3.8+
- Flask 3.0+

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python app.py
```

启动后默认监听 `0.0.0.0:5000`，便于手机端通过局域网访问。

### 3. 访问应用

- 本机访问：浏览器打开 http://127.0.0.1:5000
- 局域网访问（手机端）：确保手机与电脑同一 Wi-Fi，访问 http://<电脑局域网IP>:5000

## 使用指南

1. **新建排名**：首次进入即为空白项目；如已有历史数据，点击右上角「新建排名」并在弹窗中确认即可清空重开。
2. **设置奖项**：
   - 点击「新增奖项」添加奖项，填写奖项名称与获奖名额（正整数）。
   - 拖拽奖项卡片左侧手柄调整级别高低，**越靠前级别越高**，优先分配给排名更高的选手。
   - 点击删除图标可移除奖项（二次确认）。
3. **录入成绩**：
   - 切换到「成绩录入」标签，填写姓名与分数（支持整数与小数），回车快捷提交。
   - 已录入选手支持单条编辑（点击编辑图标修改后保存）与删除（二次确认）。
4. **生成结果**：
   - 点击「生成排名结果」一键计算，自动跳转至排名榜单。
   - 榜单展示名次、姓名、分数、获奖奖项，前三名以金/银/铜配色高亮。
   - 支持按名次 / 分数 / 姓名切换排序；长按榜单可复制排名文本。
5. **结束使用**：数据保留于本地浏览器，下次可点击「新建排名」开启新任务。

## 排名与奖项规则

- 按分数从高到低自动排序。
- 同分选手并列同名次，后续名次顺延（如两位并列第 1 名，则下一名为第 3 名）。
- 按奖项级别从高到低匹配名额，高等级名额用完后顺延至下一等级。
- **同分跨边界处理**：同分并列组整体分配同一奖项；当前奖项剩余名额不足以容纳整组时，整组顺延至下一等级奖项，被跳过奖项的剩余名额自动空置。
- 所有奖项名额用尽后，后续选手标注为「未获奖」。
- 若获奖总名额大于选手总人数，剩余名额自动空置。

## 数据与隐私

- 所有奖项配置与选手成绩仅存储于用户浏览器 localStorage，无任何数据上传至服务端。
- 服务端不部署数据库，不处理任何业务数据。
- 可通过「新建排名」清空本地数据。
